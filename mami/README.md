|            +---+--------------------v---+           |
|            |    Transport (Socket/TCP)  |           |
|            +----------------------------+           |
|                                                     |
+-----------------------------------------------------+


     ^                         +----------------------+

     +                         +----^------------+----+
                                    |            |
