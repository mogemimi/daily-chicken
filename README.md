This is my personal repo for learning CHICKEN Scheme as a hobby.

## Setup

```sh
brew install chicken
```

```sh
git submodule update --init
```

See also http://wiki.call-cc.org/platforms#mac-os-x

## Dependencies

```sh
make dependencies
```

## Build and run

```sh
make
```

## Test for Mac

```sh
make gochiusa2
```
